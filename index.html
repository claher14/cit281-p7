Project 7 – GitHub Pages Portfolio

Overview

This project involved creating a portfolio website using GitHub Pages to showcase my work from CIT 281. It includes links to my labs and projects, each hosted as a separate repository with its own Page.

Technologies Used

  • GitHub Pages
  • HTML & CSS
  • Markdown
  • Git and GitHub

What I Learned

I learned how to organize repos, set up GitHub Pages, document and present coding work, and create a visually appealing and navigable online portfolio. This project will serve as a foundation for adding future coursework and personal projects.
